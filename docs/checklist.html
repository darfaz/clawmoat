AI Agent Security Checklist

20 actionable steps to protect your machine from autonomous AI agents. Updated February 2026.

Lock down ~/.aws/, ~/.gcloud/, ~/.azure/, ~/.kube/

Secure browser data

Prevent access to browser profile directories (cookies, saved passwords, sessions)

Protect GPG/PGP keys

Lock down ~/.gnupg/

Protect crypto wallets

Lock down wallet files, seed phrases, and key stores

Limit file system scope

Restrict the agent to a workspace directory; block access to system files
